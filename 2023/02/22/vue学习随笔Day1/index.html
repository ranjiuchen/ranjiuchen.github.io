发表于2023-02-22|更新于2023-02-22|study
|阅读量:

1.vue.js 一套用于构建用户界面的渐进式 JavaScipt 框架 开发者:尤雨溪(老二次元!)

2.特点:
组件化(提高代码复用率,让代码更好维护){.vue 文件:.html,.css,.js}
声明式编码(不需要操作 DOM,提高效率)
使用虚拟 DOM+Diff 算法,尽可能复用 DOM

3.需要的 JavaScript 知识:
ES6
包管理器
原型、原型链
数组常用方法
axios
promise
……

vue.js 安装环境:

Type 1:script 标签引入
Type 2:npm 安装

创建 Vue 实例,传入配置对象

Vue 容器里的代码依然符合 html 规范,同时混入特殊的 Vue 语法

Vue容器里的代码称为Vue模板

插值语法与指令语法

数据绑定

el的两种写法

data的两种写法

vue与MVVM模型:

M:model      对应data中的数据
V:view       模板
VM:viewmodel Vue实例对象(vue真的我哭死)

data中的数据最后会成为vm实例对象上的属性

defineProperty

数据代理:通过一个对象代理对另一个对象中属性的操作(读写)

vue与数据代理:当属性被访问时,通过getter将data中的值获取给属性;
当属性被修改时,通过setter将data中的值修改。

事件绑定
